# Cloud-Security-AI-Threat-Detection-Simulation-Experiment
Experimental setup, approach, and full written report of AWS and Azure cloud environments simulating penetration testing and misconfiguration detection using native AI threat detection tools integrated with DevSecOps principles. AWS utilization of machine learning native AWS based GuardDuty and Security Hub center. Microsoft Defender used for Azure. Penetration testing performed using Nmap on live cloud servers for both providers. Snyk code vulnerability scanning reports illustrate how to remediate and patch code repositories before they are launched into a cloud environment. Part of the final project for CYBV 579-Cloud Security as part of the Masters of Science in Cyber and Information Operations degree program.
